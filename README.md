# KratosThink

## But Principal
Le site est une plateforme d'apprentissage intelligent qui utilise l'intelligence artificielle pour générer des cours personnalisés sur n'importe quel sujet...

## Design Général
- Interface épurée, moderne et intuitive
- Lisibilité et accessibilité
- Indicateurs de progression et expérience fluide

## Fonctionnalités Détailées
1. Page d’Accueil / Génération de Cours
   - Champ de saisie : "Enter a topic to learn..."
   - Bouton "Generate Course"
   - Slogan : "Learn Anything with AI", "Powered by artificial intelligence"
...
## Technologies Impliquées
- IA générative pour créer le contenu des cours
- Base de données pour suivre la progression
- Frontend interactif (React, Vue.js ou similaire)
- Backend pour la gestion des utilisateurs et des cours
- Algorithmes de recommandation pour suggérer des cours ou révisions
